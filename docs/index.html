Beat-Synced Squares

Audio-driven squares on video: onsets, ORB keypoints, Lucas–Kanade flow, neighbor edges — all timed to the beat.

This site

GitHub Pages only serves static files. Rendering needs Python, FFmpeg, and OpenCV on your machine or a server — not in the browser.

Run it locally

See the repository README for setup. In short:

  • CLI: python main.py -i clip.mp4 -o out.mp4
  • Web UI: uvicorn api.app:app --host 127.0.0.1 --port 8000 then open /
